### Browserama

Browserama is a browser detection module that uses feature detection rather than
the user agent to determine which browser is being used.

```sh
npm install browserama
```

#### Properties

- `isBlink`
- `isChrome`
- `isChromium`
- `isChromeOrChromium`
- `isEdge`
- `isFirefox`
- `isIE`
- `isOpera`
- `isSafari`
